# SpringBoot-Demo
一个SpringBootDemo工程，主要是基于SpringBoot2.0.3
## 主要的功能
### 数据持久化MySQL数据库
   主要集成Hibernate，一个简单的User类测试
### 配置Redis数据库
   配置Redis，主要配置了RedisTample
### 文件上传
   文件上传，用于文件的上传以及异常处理
### 基于MVC模型
   层次划分
### 自定义拦截器
   主要用于检查登录的拦截器，以后可以模仿写其他拦截器
### 增加thymeleaf简单用例
   增加了简单的thymeleaf用例
